# sum-count-divisible-by-4-break-continue
C program to calculate sum and count of numbers divisible by 4, while skipping multiples of 8 and stopping loop at 90 using break and continue.


# Sum and Count Divisible by 4

## 📌 Description
This program calculates:
- Sum of numbers divisible by 4
- Count of numbers divisible by 4

With conditions:
- Skip numbers divisible by 8
- Stop loop at 90

## 🧠 Concepts Used
- for loop
- if condition
- break
- continue

## 👩‍💻 Learning Outcome
Practice of loop control and conditions in C programming
